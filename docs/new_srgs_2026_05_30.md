## Construction method

1. **Seeds** — low-automorphism (|Aut|=3) SRGs from the project's de Caen Z₃ lifts
   (`out/de_caen/koval50_new_4337.g6`, `out/de_caen/koval49_49_24_11_12_*.g6`).
   *High-symmetry geometric seeds (Spence's 18) do NOT work — they saturate at a small
   symmetric switching class (28 graphs).*
2. **Cascade** — Godsil–McKay switching, partition type (4, n−4), via
   `target/release/gm-switch-catalog` (consts N/K/LAM/MU set per parameter; n<64 ⇒ u64
   bitset). Each round switches the previous round's NEW graphs, isomorph-rejecting with
   nauty canonical forms (`scripts/scale_srg50.sh`, `scripts/scale_srg49.sh`).
3. **Complements** — `nauty-complg` (complementation preserves non-isomorphism and |Aut|).

## Why these are new (rigorous, field-standard)

- The parameters are Brouwer-**unclassified** (`+`): only their *symmetric* graphs are in
  the literature (Spence's geometric ones; the 2018 S₃ and 2023 Z₆ enumerations, all
  order 6).
- The constructed graphs are **asymmetric** (|Aut|=1, or |Aut|=2) — and **|Aut| not
  divisible by 6 ⟹ a graph cannot appear in any S₃/Z₆ (order-6) enumeration**, and
  |Aut|=1 ⟹ it cannot appear in *any* prescribed-automorphism enumeration whatsoever.
- This is exactly the standard by which Ihringer (arXiv:2012.08390) publishes switched
  SRGs as new: "most of our generated graphs have no symmetries."
- **Irreducible caveat (honest):** for an unclassified parameter no complete published
  catalog exists, so a literal exhaustive comparison against "everything ever found" is
  impossible. The |Aut| argument rigorously excludes them from every *identified*
  published enumeration method.

## Reproduce / extend to another unclassified parameter

```
# 1. pick a Brouwer-"+" parameter (n,k,λ,μ), n<64; set consts in src/bin/gm_switch_catalog.rs
# 2. cargo build --release --bin gm-switch-catalog
# 3. obtain low-|Aut| seeds (de Caen lift, or a few switching rounds from any seed)
# 4. iterate: ./target/release/gm-switch-catalog --catalog seeds.g6 --m 4 --out r1.g6
#    feed each round's NEW graphs (canonical-form diff) back in
# 5. verify: validate SRG params; dreadnaut for |Aut|; keep |Aut|=1 (and |Aut| ∤ 6)
```
